# HNK / React JS Project Fully Responsive

## Installation

```bash
git clone https://github.com/halenisakus/react-website.git
cd react-website
npm install
npm start
```

## Used Technologies

- Responsive Design CSS Library <br/>
- Node.js <br/>
- React Hooks <br/>
- React Router <br/>
